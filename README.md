# Shetkari Setu 🌾

Shetkari Setu is a farmer-to-customer direct marketplace platform.

## Project Goal
- Help farmers sell products directly to customers
- Remove middlemen
- Support English, Marathi, Hindi
- Provide contact desk for farmer assistance

## Modules
- Farmer
- Customer
- Admin
- Product Management
- Order & Payment

## Technologies (Planned)
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- Payment: Razorpay
